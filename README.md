# 🏥 Clínica Parginalli

Projeto de um site institucional desenvolvido para uma clínica fictícia, com o objetivo de praticar HTML e CSS na construção de páginas estruturadas, organizadas.

---

## 📌 Sobre o projeto

O site da **Clínica Parginalli** foi criado como parte dos estudos em desenvolvimento web, focando na criação de múltiplas páginas e na organização de layout utilizando HTML e CSS.

O projeto simula um ambiente real de uma clínica, apresentando informações essenciais como serviços, horários de atendimento e formas de contato.

---

## 🧩 Páginas do site

* **Página Principal** – Apresentação da clínica
* **Sobre a Clínica** – Informações institucionais
* **Horário de Atendimento** – Tabela com especialidades e horários
* **Contato** – Informações de contato, localização e formulário

---

## 💻 Tecnologias utilizadas

* HTML5
* CSS3

---

## 🎯 Objetivo

Este projeto tem como objetivo:

* Praticar a estruturação de páginas HTML
* Trabalhar com organização de layout
* Criar navegação entre páginas
* Desenvolver um site simples e funcional
